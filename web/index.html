端点列表

POST /api/product/lookup

查询商品信息。输入ASIN或商品URL，返回标题、价格、评分、图片、卖家等。

Body: { "asin": "B0D1XD1ZV3" } 或 { "url": "https://amazon.com/dp/B0D1XD1ZV3" }
POST /api/product/price-history

获取商品价格历史。默认返回30天价格走势。

Body: { "asin": "B0D1XD1ZV3", "days": 30 }
POST /api/keyword/search

关键词搜索商品。返回Top 20结果。

Body: { "keyword": "wireless earbuds", "platform": "amazon" }
POST /api/keyword/suggest

关键词建议。输入种子词，返回相关长尾词。

Body: { "seed": "bluetooth speaker", "platform": "amazon" }
POST /api/competitor/monitor

添加竞品监控任务。

Body: { "productId": "B0D1XD1ZV3", "platform": "amazon", "alertOn": ["price_change"] }
GET /api/competitor/alerts

获取竞品变动提醒。

Query: ?limit=50&unread=true
GET /api/usage

获取API用量统计。

GET /api/health

健康检查（无需认证）。
